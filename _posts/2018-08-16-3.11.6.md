🎉 New

🍰 Enhancements
 * Alert messages translated to swedish
 * Users can now choose labels for a task before creating it
 * New filter option for tasks - "Created By Me"
 * The list of most recently visited boards will only show boards that are in the same organization/construction project
 * The search function will now organize the boards under the relevant organization/construction project

🐛 Fixes
 * Task context menus sometimes ended up hidden behind the right-hand side panel
 * Filter that only affects questions should only be included for boards that use questions
