🎉 New
  * Question now only have one input field called Post that replaces Answer and Comments.
  * You can now mark a Post as answer and are allowed to change that at any time.
  * Introduce 16 week view as an option
  * New filters available! Text search and some specific for Construction questions.

🍰 Enhancements
  * Horizontal scroll always visible in list
  * Column header row visible while scrolling in list
  * Restyling of Manage Boards page to minimize risk of accidentally archiving organization

🐛 Fixes
  * Fixes bug where notes where overlapping while side panel was open
