🎉 New

🍰 Enhancements

🐛 Fixes
 * Major performance improvements for large construction projects.
   * Navigating/Scrolling through dates should feel a lot smoother.
   * Filtering on row properties such as Locations and Trades will help reduce the load on boards for large projects.
 * Performance fix for sites running Initiative or Discipline board variants. Changing ownership of a row could cause a short but drastic slowdown in data query speed
 * Performance fix for sites using the job behavior. Accepting a job could cause a short but drastic slowdown in data query speed
 * PPC/TA for weeks was not showing properly after new year.
 * The filter dropdown menu is showing the "reset filter" button as disabled when no filter is active.
 * When double clicking really fast on "today" or a specific resolution the board would sometimes show up empty.
